Project Overview

What is NFX Chain?

NFX Chain is a next-generation blockchain platform that combines cutting-edge technologies to create a secure, scalable, and intelligent decentralized network. It features quantum-resistant cryptography, AI-driven governance, and a hybrid consensus mechanism called Proof of Exponential Security (PoES).

Quantum Resistant

SHA-3 based cryptography protected against quantum attacks

AI Governance

Neural networks validate blocks and manage network parameters

Hybrid Consensus

PoW + PoS + PoES for optimal security and efficiency

Smart Contracts

JavaScript VM for flexible contract development

Data Flow

  1. Transaction created in Wallet or via API
  2. Broadcast through P2P network to hypernodes
  3. Validation by AI governance and consensus rules
  4. Block creation by selected minter (PoW/PoS/PoES)
  5. Verification by network consensus
  6. Chain addition and state update in LevelDB

Installation & Building

Ubuntu 22.04+

Terminal 
# Install dependencies
sudo apt update
sudo apt install -y build-essential cmake git \
    qtbase5-dev qt5-qmake qtbase5-dev-tools \
    libleveldb-dev libssl-dev libboost-all-dev \
    wget curl

# Install Go 1.19+
wget https://go.dev/dl/go1.19.13.linux-amd64.tar.gz
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.19.13.linux-amd64.tar.gz
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc

# Clone and build
git clone https://github.com/NFXChain/nfx-chain.git
cd nfx-chain

# Build nfx-core
cd nfx-core
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
cd ../..

# Build nfx-go
cd nfx-go
go mod download
go build -o nfx-node ./api
cd ..

# Run node
./nfx-go/nfx-node --config config.toml

Code Examples

Using the Go API

main.go 
package main

import (
    "fmt"
    "log"
    "github.com/NFXChain/nfx-go/api"
)

func main() {
    // Connect to local node
    client := api.NewClient("http://localhost:8332", "nfxuser", "yourpassword")

    // Get blockchain info
    info, err := client.GetBlockchainInfo()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Chain: %s\n", info.Chain)
    fmt.Printf("Blocks: %d\n", info.Blocks)

    // Send transaction
    txID, err := client.SendTransaction("NFX_ADDRESS", 1.5)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Transaction sent: %s\n", txID)
}

Using C Bindings

example.c 
#include <stdio.h>
#include "nfx_bindings.h"

int main() {
    // Initialize node
    nfx_node_t *node = nfx_node_create();
    nfx_node_init(node, "config.toml");

    // Connect to network
    nfx_error_t err = nfx_node_start(node);
    if (err != NFX_OK) {
        printf("Failed to start node: %s\n", nfx_error_string(err));
        return 1;
    }

    printf("Node started successfully!\n");

    // Get balance
    uint64_t balance;
    err = nfx_get_balance(node, "NFX_ADDRESS", &balance);
    if (err == NFX_OK) {
        printf("Balance: %lu\n", balance);
    }

    nfx_node_destroy(node);
    return 0;
}

Configuration File (config.toml)

config.toml 
# NFX Chain Node Configuration

# Network
datadir = "/home/user/.nfx/data"
logfile = "/home/user/.nfx/logs/nfx.log"
loglevel = "info"

[network]
testnet = true          # Use testnet (set false for mainnet)
p2p_port = 18333
rpc_port = 18332
maxconnections = 50

[consensus]
consensus = "poes"     # pow, pos, or poes
powdifficulty = 1
posminstake = 1000

[rpc]
rpcuser = "nfxuser"
rpcpassword = "securepassword"
rpcallowip = "127.0.0.1"

[performance]
dbcache = 256          # MB
maxmempool = 128       # MB
